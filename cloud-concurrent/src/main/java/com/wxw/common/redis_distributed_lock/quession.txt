
1. 遗留的问题：没有解决锁的自动续期（不断的延长锁的有效期）问题
   解释说明：在设置锁过期时间时，需要去预估业务耗时时间，如果锁的过期时间能根据业务运行时间自动调整，那使用的就更方便多了
   解决办法：使用已经解决这个问题的 Redisson框架，通过watchdog 实现锁的自动续期


2. 相关面试题
  1. Redis锁的过期时间小于业务的执行时间该如何续期？（业务还没执行完锁就过期了）
     解决办法：
      （1）Redisson 使用watchdog 实现分布式锁的自动续期
      （2）自定义一个守护线程，进行续约（https://juejin.cn/post/6844903764868988936）
